# Web-Based PE File Analyzer

A lightweight static analysis tool built with **Python** and **Flask**. This application parses Windows Portable Executable (PE) files (`.exe`, `.dll`, `.sys`) to visualize internal headers, sections, and identify potential security anomalies without executing the code.
